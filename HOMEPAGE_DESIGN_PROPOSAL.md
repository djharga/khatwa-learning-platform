## 🎯 التفاصيل لكل قسم

### 1. Hero Section (القسم البطولي) ⭐

**الهدف**: جذب الانتباه وإيصال الرسالة الرئيسية

**المكونات المقترحة**:
```tsx
┌─────────────────────────────────────────────────┐
│  [Badge] منصة التعليم المهني الرائدة          │
│                                                 │
│  # خطى: رحلتك نحو التميز في المحاسبة          │
│  والمراجعة الداخلية                            │
│                                                 │
│  وصف مختصر (سطر أو سطرين)                      │
│                                                 │
│  [ابدأ التعلم الآن] [شاهد العرض التوضيحي]     │
│                                                 │
│  [صورة/فيديو تعريفي]                          │
└─────────────────────────────────────────────────┘
```

**الألوان**: تدرج لوني من primary إلى accent
**التأثيرات**: Animation خفيفة للعناصر
**CTA**: زر واضح وكبير للبدء

---

### 2. Trust Indicators (مؤشرات الثقة) 🏆

**الهدف**: بناء الثقة من خلال الأرقام والشهادات

**المكونات**:
```tsx
┌─────────────────────────────────────────────────┐
│  📊 10,000+      📚 500+        ⭐ 4.9/5        │
│  طالب مسجل      دورة متاحة    تقييم المتعلمين │
│                                                 │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                 │
│  [شعارات الشهادات والاعتماد]                  │
│  CIA • CMA • IIA                                │
│                                                 │
│  [شعارات الشركاء - إن وجدت]                    │
└─────────────────────────────────────────────────┘
```

**التصميم**: Cards بسيطة مع أيقونات كبيرة
**Animation**: Counter animation للأرقام

---

### 3. Features (المميزات الرئيسية) ✨

**الهدف**: عرض القيمة المضافة للمنصة

**التصميم المقترح**:
```tsx
Grid 3 Columns (Desktop) / 2 Columns (Tablet) / 1 Column (Mobile)

┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  📚          │ │  🎓          │ │  🏅          │
│  دورات       │ │  شهادات      │ │  متابعة      │
│  احترافية    │ │  معتمدة      │ │  التقدم      │
│              │ │              │ │              │
│  وصف مختصر   │ │  وصف مختصر   │ │  وصف مختصر   │
└──────────────┘ └──────────────┘ └──────────────┘

┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  💬          │ │  📱          │ │  🔒          │
│  دعم فني     │ │  تعلم       │ │  محتوى       │
│  24/7        │ │  محمول      │ │  محمي        │
│              │ │              │ │              │
│  وصف مختصر   │ │  وصف مختصر   │ │  وصف مختصر   │
└──────────────┘ └──────────────┘ └──────────────┘
```

**المميزات المقترحة**:
1. 📚 **دورات احترافية** - محتوى عالي الجودة
2. 🎓 **شهادات معتمدة** - اعتماد من جهات معروفة
3. 🏅 **متابعة التقدم** - تتبع مستمر للإنجازات
4. 💬 **دعم فني 24/7** - مساعدة متاحة دائماً
5. 📱 **تعلم محمول** - وصول من أي جهاز
6. 🔒 **محتوى محمي** - أمان عالي للمحتوى

### 4. Popular Courses (الدورات الأكثر شعبية) 📚

**الهدف**: عرض أفضل الدورات وإثارة الاهتمام

**التصميم**:
```tsx
┌─────────────────────────────────────────────────┐
│  ## الدورات الأكثر طلباً                        │
│                                                 │
│  [Tab: جميع | محاسبة | مراجعة | تدريب]         │
│                                                 │
│  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐  │
│  │ Course │ │ Course │ │ Course │ │ Course │  │
│  │ Card   │ │ Card   │ │ Card   │ │ Card   │  │
│  └────────┘ └────────┘ └────────┘ └────────┘  │
│                                                 │
│  [عرض جميع الدورات →]                         │
└─────────────────────────────────────────────────┘
```

**Course Card يجب أن يحتوي على**:
- صورة الدورة
- العنوان
- المدرس
- التقييم (نجوم)
- السعر/مدة الدورة
- Badge (جديد، شائع، معتمد)
- زر "عرض التفاصيل"

### 5. How It Works (كيف يعمل) 🎯

**الهدف**: تبسيط عملية البدء وإزالة التعقيد

**التصميم**: خطوات أفقية مع أيقونات
```tsx
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│   1️⃣     │ →  │   2️⃣     │ →  │   3️⃣     │ →  │   4️⃣     │
│  سجّل    │    │  اختر    │    │  ابدأ    │    │  احصل   │
│  حساب    │    │  دورة    │    │  التعلم  │    │  شهادة  │
│          │    │          │    │          │    │          │
│  وصف     │    │  وصف     │    │  وصف     │    │  وصف     │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```

**الخطوات**:
1. **سجّل حسابك** - إنشاء حساب مجاني
2. **اختر دورتك** - تصفح واختيار الدورة المناسبة
3. **ابدأ التعلم** - وصول فوري للمحتوى
4. **احصل على شهادة** - إتمام الدورة والحصول على شهادة

## 🎨 التصميم البصري المقترح

### الألوان:
- **Primary**: #0066FF (أزرق احترافي)
- **Accent**: #818CF8 (بنفسجي)
- **Success**: #22C55E (أخضر)
- **Background**: #FFFFFF / #F7FAFC
- **Text**: #1E293B

### Typography:
- **العناوين**: Cairo Bold, 32px - 48px
- **النصوص**: Tajawal Regular, 16px - 18px
- **الوصف**: Tajawal Light, 14px - 16px

### Spacing:
- **بين الأقسام**: 64px - 96px (desktop)
- **داخل القسم**: 24px - 32px
- **بين العناصر**: 16px - 24px

### Animations:
- **Fade In**: عند التمرير (scroll-triggered)
- **Slide In**: للعناصر الجانبية
- **Scale**: للأزرار عند hover
- **Duration**: 200ms - 400ms (خفيف وسريع)

## 📱 Responsive Design

### Mobile (< 768px):
- Hero: عمود واحد، صورة أعلى
- Features: عمود واحد
- Courses: عمود واحد أو عمودين

### Tablet (768px - 1024px):
- Hero: عمود واحد، صورة أسفل
- Features: عمودين
- Courses: عمودين أو ثلاثة
- Testimonials: عمودين

- Hero: عمودين (نص | صورة)
- Features: ثلاثة أعمدة
- Courses: أربعة أعمدة
- Testimonials: ثلاثة أعمدة
